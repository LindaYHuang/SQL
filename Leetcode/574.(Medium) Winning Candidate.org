 Vote
 #+BEGIN_EXAMPLE
+-----+--------------+
| id  | CandidateId  |
+-----+--------------+
| 1   |     2        |
| 2   |     4        |
| 3   |     3        |
| 4   |     2        |
| 5   |     5        |
+-----+--------------+
#+END_EXAMPLE

output
#+BEGIN_EXAMPLE
+------+
| Name |
+------+
| B    |
+------+
#+END_EXAMPLE

Notes:
You may assume there is no tie, in other words 
there will be at most one winning candidate.


-Solution
#+BEGIN_SRC python
Select Name from Candidate
Where ID = (SELECT CandidateId FROM Vote 
            GROUP BY CandidateId 
            ORDER BY count(id) Desc
            Limit 1)
#+END_SRC
